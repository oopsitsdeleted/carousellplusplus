# Carousell++
Removes sponsored, shoutout and spotlight posts when searching on Carousell, allowing you to find what you are actually looking for

To use it for yourself, install the [Tampermonkey extension](https://www.tampermonkey.net/) in your browser, then follow the prompts [here](https://greasyfork.org/en/scripts/557565-carousell)
